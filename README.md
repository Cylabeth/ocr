# RodexID

Responsive webApp based in OCR for personal documentation recognition. Ironhack WDFT final project.

- React Camera Pro
- S3
- AWS SDK
- Textract
- Javascript
- MongoDB
- Boostrap
- Express.js
- Node.js
- Passport
